# Pizzeria
Программа для заказа пицце делающая упор на простату использования

## Возможности
Корзина с товарами на покупку
Виртуальный кошелёк
Цепи Маркова для образования капчи
Регистрация пользователя
Цветной вывод
Система лайков
Взрослое и детское меню
Возможность добавлять свои пицце
Учёт ингредиентов
Меню и скидки

## Установка
У вас должны быть установлен python <b>(желательно 3.13)</b>

1. Клонирование репозитория 

```git clone https://github.com/ProgrammsStorge/Shops.git```

2. Переход в директорию Shops

```cd Shops```

3. Установка зависимостей

```pip install -r requirements.txt```

4. Запуск скрипта

```python main.py```
